# Contributing to Parkinson's Memory Coach

Thank you for considering a contribution. This project is personal support for Jim's father with Parkinson’s and is open for collaboration.

## Ways to Contribute
- Feedback on real needs (Issues).
- Code, tests, STT/vision/memory improvements.
- Documentation and testing with MockDeviceKit.
- Respectful promotion in relevant communities.

## Development Setup
1. git clone https://github.com/jodonnel/parkinsons-memory-coach.git
2. Configure local.properties (Meta token etc.).
3. ./gradlew build
4. Run debug with MockDeviceKit.

See docs/CONCEPT.md for vision.

## Code of Conduct
Respectful, constructive, privacy-aware. This affects families.

Questions? Open an Issue.
